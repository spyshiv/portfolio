Writing Bash Script is a good habit for a developer. If you want to be an ethical hacker then you must love script writing.


Guideline For Writing First Script
offline_pinopen your Linux terminal (hit Ctrl+Alt+T )
offline_pincreate a file with extension .sh (e.g. first_script.sh)
offline_pinwrite following code to that file
#!/bin/bash
echo "Hello World!";

Note: First line is the path to where bash is installed in your computer system. And Second line is simple code to print “Hello World!”

offline_pinSave the file. Now file should have executable permissions so you have to give execute permission to that file
sudo chmod +x first_script.sh
offline_pinrun the script by typing to terminal the following line 
./first_script.sh

Cheers!!
